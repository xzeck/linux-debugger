# linux-debugger

linux-debugger is a small gdb-like debugging tool for linux

## Functionalities:
- Add break point at:
    - address
    - function
    - line
- register
    - read
    - write
    - dump
- memory
    - read
    - write
- step in
- step over
- step out
- backtrace
- get all symbols
- get all variables
- get all functions
